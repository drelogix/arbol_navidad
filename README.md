# Árbol de Navidad Animado en JavaScript

Este proyecto muestra cómo crear un árbol de Navidad animado utilizando HTML, CSS y JavaScript puro. El árbol se dibuja completamente dentro de un elemento canvas y utiliza una espiral de luces que asciende desde la base hasta la punta, generando un efecto visual fluido y realista.

---

## Características

- Renderizado mediante Canvas 2D API.
- Espiral de luces animada con velocidad ajustable.
- Control de color para las luces.
- Botón para encender y apagar la animación.
- Estrella y árbol generados con cálculos geométricos, sin imágenes externas.
- Proyecto sin dependencias externas.

## Cómo ejecutarlo

1. Descarga o clona el repositorio.
2. Abre el archivo index.html en cualquier navegador moderno.
3. Utiliza los controles para ajustar la animación según prefieras.

No requiere instalación adicional ni servidor.

---

## Conceptos aplicados

- Canvas 2D API
- Coordenadas polares
- Generación de trayectorias en espiral
- Animación frame-by-frame con requestAnimationFrame
- Gradientes para simular volumen
- Efectos de brillo mediante shadowBlur
- Transformaciones y cálculo geométrico

---

## Mejoras posibles

- Añadir un fondo con efecto de nieve.
- Crear una animación pulsante en la estrella.
- Implementar un modo de color automático.
- Agregar decoraciones adicionales alrededor del árbol.
